# fs-lab-core-api-go

Minimal Go backend built for an **API-first runtime comparison**
between **Go, Node.js, and Python**.

The service exposes identical endpoints and response contracts
to allow fair comparison of startup time and request latency.

---

## Endpoints

- `GET /health`
- `GET /meta`

---

## Run locally

```bash
go run ./cmd/server
```
